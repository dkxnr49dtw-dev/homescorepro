# HomeScorePro React Application

This is the React version of HomeScorePro, built with Vite, React Router, and Framer Motion.

## Prerequisites

- Node.js 18+ and npm

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```
